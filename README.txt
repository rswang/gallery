Instructions

Import gallery.sql file using phpmyadmin. It will create the necessary database and tables.
To upload photos, visit the admin folder using your favourite browser. Try uploading the example photos in the example-photos folder and thats it.
Remember it you need different thumbnail size..change the width parameter and also don't forget to change the css for the container, thumbnail and info divs.
